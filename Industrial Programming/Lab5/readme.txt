  Использовать классы : Matcher, Pattern, PatternSyntaxException.
  
  Общее задание:
1. Данные считать из файла (> =10 тестовых строк).
2. Записать результаты в файл. Использовать классы : FileReader, FileWriter, BufferedReader,
BufferedWriter.
3. Можно использовать коллекции (класс ArrayList).

 Написать регулярное выражение определяющее является ли данная строчка временем в
формате ss/mm/hh, ss:mm:hh или ss-mm-hh

Пример правильных выражений:
01/01/12
59/59/23
00-00-24
Пример неправильных выражений:
61/01/12
01/61/12
01/33/25
1/33/22
59/59-23
19/09-23
00-00-223
