  Процесс Parent:
Ввести размер массива, ввести элементы массива;
Для вариантов ввести необходимые дополнительные значения (K);
Формирует командную строку, которая содержит информацию об размерности массива, элементах и т.д.
Для консоли дочернего прооцесса устанавливает визуальные настройки: установить любой цвет текста (не белый) для Сhild.
Запускает дочерний процесс Child, которому через командную строку передается информация об размерности массива, элементах и т.д. (согласно варианту);
  Процесс Child:
Выполнить суммирование элементов итогового массива до заданной позиции K. Полученный массив вывести. Тип элементов - вещественные числа.
